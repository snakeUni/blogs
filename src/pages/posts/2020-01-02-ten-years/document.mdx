# 2010

2010 年我还是一个高中生，就读于 `江苏省沭阳高级中学`。在江苏省竞争是很激烈的，所以我们每周都要做好多的试卷，2010 年是我就读于高中的最后一年，高三是忙碌的，每天早上
`6` 点必须起床去上早自习，晚上大约是 `10:10` 下晚自习，每天都是重复着同样的事情。但是高三也是我最难忘的一年，因为那一年大家都在为自己理想的大学去奋斗。`10:10` 分
下晚自习后很少有人到家就会直接睡觉，我在回到家后，都会看书到晚上 `00:00` 才会入睡。在现在看来这是一件不可思议的事情，每天只睡 6 个小时左右。但是在那个时候经常有
同学学习到夜里 `2-3` 点才会睡觉。所以别人都在努力，不会抱怨，我也不应该去抱怨。只要在高考的时候考的好，才能进入好的大学。

那一年也是幸运的一年，因为高三很多学校都会补课，假期放假都会变的很少。但是我们这一届是没有补课的，从高一到高三每个寒假和暑假都享受了整个假期时间，这是一件令人振奋的一件事情。

我高三最擅长的是数学，语文是真的很差，我是理科生，在江苏省理科生的数学满分是 200 分(附加题有 40 分)，一般数学稍微不错的，附加卷都可以拿到 30 分。对于主卷的 160 分，
我基本每次都能保证 140 以上。英语满分是 120 分，每次英文考试基本都拿到 90 分以上，超长发挥的时候能拿到 105 以上。但是对于 160 分的语文，我从没有考过 120 分，
甚至 115 都没有考过，每次基本上都在 100-110 之间波动。我自认为我的语文是不错的，但是不知道为什么每次都会考那么少的分，可能是因为`我不懂鲁迅`吧。

2010 没想过其他的事情，一心都在学习上, 但是也一直没有改变过，不好的科目我在怎么努力还是一样。

# 2011

这一年迎来了一生中最重要的时刻就是`高考`, `2011-06-07 ~ 2011-06-09` 这三天是高考的时间，我对一件事记忆犹新就是高考的前一天我还发烧了，可能是由于紧张过度吧。

江苏省的高考制度是很特殊的，我们在高二会进行小高考，因为我是理科生，所以在高二的时候考了`政史地生`四门小科。在高三的时候，这四门就可以不用学习了，完全一心在`语数外理化`，
但是请注意，`理化`在高考中也是不记与总分的，所以总分只有`语数外`，三门总分是 `480` 分。高考后需要等到大约 20 天左右，高考的分数才会出来，那时候一本线，
二本线的分数都会出来。所以高考分数没出来的 20 天是最幸运的。因为可以不用学习了，可以肆无忌惮的和朋友去玩。

高考分数出来后，幸运的是我通过了一本线，并且还高出了不少分。所以我们一家都很开心。剩下的很多时间都在选择学校，因为填写志愿比高考还要让人紧张，如果第一志愿上不了那只能上
第二志愿的学校了。但是有可能第二志愿的学校不一定还会要你，所以填写志愿是各格外重要的。

高考我的理化是 `B+` 和 `B`, 如果要想去一本必须要达到`双B`以上。幸运的是我通过了。但是我这样的等级在江苏省是很难去好的一本大学的，虽然我的语数外分数足够，且多了不少。
但是理化的等级是不够的，好的一本需要 A 甚至 `A+`。最终我放弃了江苏的院校，选择来到上海。因为上海这边的院校很多都是`双B`就可以了。

我来到了[华东师范大学](https://www.ecnu.edu.cn/)，对于专业的选择，我自己也是毫无头绪的。就随机的选择了一个专业`软件工程`，从此我开启了我的程序员道路。

感谢我的母校 - [华东师范大学](https://www.ecnu.edu.cn/)

# 2012 - 2013

这一年我大二了，大学的生活让我沉沦了，我疯狂的沉迷于游戏中无法自拔 - `征途`, 我花了大量的时间以及金钱在游戏中。每天上课的时候课也不听，玩手机打游戏。让我记忆游戏的
就是 `计算机基础`，在期末考试中，这个学科我拿到了 `C`, 因为这门课我的期末成绩是 `60` 分，可能这个分数也是老师对我的施舍吧。这一年是比较平淡的一年，学业上没有
一点的进步，反而差一点挂科了。（我大学四年没有挂过科）

# 2014

在这一年我开始了实习，因为在学校的时候总是打游戏，对于专业课的知识都没有去学习。以至于在找实习的时候不知道该找什么样的实习，没有方向，没有知识基础。就像断了线的
风筝一样，风在哪，我就在哪。那时候希望只要有公司收留我，我就留下来好好实习。那一年我去了`携程`实习。实习的工作是`测试工程师`。当时在携程的时候测试工程师只是对
功能进行测试，很少会涉及到内部代码甚至自动化的测试。

我实习差不多 4 个月就离职了，在实习的时候学习了职场的一些知识，我相信这对以后是有帮助的。2014 年 10 月我辞职了，因为我想考研。很多人会问我为什么打算考研了，
还去学习呢？其实我自己也不知道，考研也是临时来的兴趣(我是跨专业考研，打算考上财的金融系)。我从 9 月份开始买考研的书籍看，数学、英语、政治以及金融方面的书。

好啦，结果可想而知，考研我失败了。如果这要能成功的话，那我就是天才了。我没有尽力，一切都是想到什么去做什么，完全没有去计划过。我现在想到那个时候的自己也的都会
感觉羞愧，但是没有遗憾，因为我玩了很多人没有玩的东西，知足常乐(我这里只是陈述我的生活，并不是呼吁大家和我一样大学去玩而忽略学习)。

# 2015

这一年所有的同学都已经在找工作了，或者已经有 offer 了。而我却在弥补学分，我还差了不少学分才能毕业。所以这一年我也选了很多课，幸运的是我的学分够了，我也可以顺利毕业
了。在 6 月份很多同学都马上去参与工作当中，而我还在找工作。因为我的专业知识真的很一般，去面试了很多大公司，根本没有一家大的公司愿意要我。我在 6 月份基本上每两天就有
一场面试。我曾经一度想过放弃，后了终于有了一家公司愿意收留我，是一家证券公司，在里面是做测试工程师。

刚毕业的我还是很痴迷于游戏，每天晚上回家都要玩游戏。曾经有一段时间是完全没有钱了，每周末还去兼容。因为我毕业了，不能在向家里人开口要钱了。在新的公司不是很忙，因为上海
的是分公司，开发人员也就 3 个左右，测试人员两个(包括我)。所以在 2015 年都在进行测试工作，我们的 web 开发都是外包的，我们什么都不用做。每天下午 5 点准时下班，有一次
钥匙忘记放在了公司，去公司看见外包的同事 7 点还在加班,当时在想怎么那么辛苦呀。都 7 点了还在上班。

因为在测试的时候需要了解相应的功能，我经常到那个外包的 web 开发人员旁问相关的功能。看到他开发出来那些页面觉得真的很酷，在这一年我爱上了 web 开发。 `javascript` 我来啦

# 2016

在公司的第二年，因为公司的事情不多，所以我基本上天天都在学习 web 开发。2015 年末，我买了 `HTML5 开发`, 我花了不到 1 个月的时间把很厚的一本书完全看完了，并且写出了一个
完整的页面。当时兴奋的睡不着，喊同事来看，快看我实现的页面，酷嘛。好吧，现在我知道那只是一个静态页面而已。一点交互都没有

由于我们的外包人员有时候不在，所以 web 出 bug 了没有人去改，而我就承担了这个任务。同事把他买的 `锋利的 jquery` 借给了我。从此我迷恋上了 `javascript`, 我开始学习
`jquery` 的每一个 api, 学习它的用法。渐渐的我们这边的 web 开发任务都到了我这里，我也慢慢的从测试走向了开发。

在 16 年年中，我们公司搬到了浦东，安卓开发和 ios 开发都走了。我开始承担了 ios 的部分工作，其实主要工作就是 package ios 包，现在想想也是蛮好玩的呀。后来我们新来了
ios 开发工程师，一些 bug 的修复以及 package 都不是我做了。那时候公司需要做网站，但是我不会开发 java, 于是公司招了两名`java web`开发来一起开发网站。从那时候起，
我又开始学 java 和他们一起开发后端以及相应的前端代码。我现在还记得我们的渲染都是在后端进行的。模板是 `ftl` 。也是我做的第一个[网站项目](http://www.lxzq.com.cn/)
并且真正部署到生产环境的。

后来我独自负责了我们公司[投资者教育网站](http://edu.ykzq.com/)的开发工作(仔细看，里面都是 jquery), 这个网站从前端到后端到数据库都是我一个人负责的。在负责这个项目的过程中，学习到了很多知识，
在此过程中也让我成长了许多。

# 2017

2017 年是比较轻松的一年，在 2016 后半年，我们又开发了一个模拟证券投资的项目，那时候我开始接触了 mvvm 框架 - `angular1`。在使用 `angular1` 开发的时候看到 web，
还能这样开发，感觉对我的冲击是很大的。所以在 2016 结束这个项目后，我开始疯狂的学习各种 `javascript` 的知识以及各个框架。那时候就是 `Vue`

这一年有个小插曲就是我想跳槽，又去了携程面试，面试官问我：原型你知道嘛？ 我心里的想法是原型是什么？js 有这样的东西吗？。如我所料，我并没有得到这份工作。这是
17 年上半年的事情。我经历过这次后，开始学习 `javascript` 基础，到 js 的红宝书以及各种 `js` 书籍。我在这一年时间看的各种专业书籍包括服务端等等。快到 20 本了。
现在我的电脑里仍然有这样书籍的 pdf。

也是在这一年，我接触到了 `Vue` 和 `React`，一个让我看了就迷上的框架 - `React`。我非常喜欢 `React`， 也是第一次见识到原来前端代码可以这样来写。因为在公司没有
这样的项目让你用框架去重构，并且公司也不会轻易的让你去尝试。所以我开始自己启项目，开始了我的 github 之旅。开始自己尝试写各种简单的组件。同时也学习了 `Redux`,
惊叹于前端代码也能实现`时间旅行`。

# 2018-2019

在 2018 我做了一个很重要的决定。因为我觉得在老公司已经无法在进步了，没有人和我交流技术， 所以我毅然而然的决定离职。我 5 月份离开了，来到了新的公司 - `拼多多`,
选择去一个公司，我最看重的是这个公司给我带来的成长是多少。只要能给我带来足够的成长，对我来说就是足够的。现在发现，来多多是我明智的选择，当时拿了好几个 offer,
但是我看中多多的技术氛围，所以我来到了这里，这里也没让我失望。感谢多多给了我更多学习的机会以及让我能更加快速的成长。

我的信仰是 `蛇`， 为什么是这个动物？因为它可以隐忍，总能在最适合的时机给人致命一击，遇强则强。
